# 1. 属性操作

- attribute
  - attr()
    -    console.log($('ul').attr('name'))
  - removeAttr()





- property
  - prop()
    -    console.log($("ul").prop("name"))//获取不到，因为不是属性，是标签里面的内容
    - 但id可以,对象中有id
    - 添加多个,参数可以是对象
  - removeProp()
    - 只能删除自定义的属性,内置的属性不能删除





- attrbute和property的区别
  - attr是标签上面的属性
    - 给其加个属性,标签上可以看到这个属性

  - 而prop是标签这个元素对象上的属性
    - 给其加标签上没有,要将其转为对象,加的属性在其里面
      -  console.log("%O", document.querySelector('ul'))//%O的O必须大写






- data
  - data()
    - 获取现在缓存上面找,再去html的标签找
    - 缓存可以在相应对象找到
  - removeData()

# 3. jQuery里面的事件操作

- 事件初体验
  - 监听事件

    - $('ul').on(function(){})
    - $('ul').click(...)
    - $('ul').mouseenter(fuction(){})

  - 取消事件

    - ```
      $('ul').on('click',function(){  
      	$('ul').off()//取消ul所有事件
      	$('ul').off('mouseenter')//取消单个事件
      })
      ```
    - 

  - 自动触发事件

    - ```
      $('ul').trigger('click')//自动触发click事件
      ```

      

- click和on的区别

  - ```
       // $('ul').on(function() {
          //   console.log(this)
          // })没效果,必须要加参数
          // $('ul').on('click', function() {
          //   console.log(this)
          // })
    ```

  - click是on的简写,底层都是addEventListener

  - ````
     // 区别主要在on上
          //1 on支持命名空间(click.ul)
          //2.有个selector的参数，可以指定事件源
      
          $('ul').on('click.ul',".li-4",function() {
            console.log('li-4被点击了');
          
          })
          $('ul').click(function(){
          	$('ul').off('click.ul')
          })
    ````

- 事件回调函数中的this

  - ```
    $('ul li').click(function() {
            console.log($(this).addClass('active'))
          })
    ```

    

- 事件冒泡

  - 支持但不支持捕获,底层没有实现

- 事件对象
  - 原生事件对象
  - jQuery事件对象
    - 里面可以拿到原生的事件对象(event.origineEvent)

- 阻止事件

  - ```
    $('a').click(function(e){
    	//e.preventDefault()//不跳转
    	console.log('点击a元素')
    	e.stopPropagation()//阻止事件的冒泡
    })
    
    ```

  - 

- 事件委托

  - ```
       //事件委托:本质上是利用事件冒泡的原理,在父元素上监听子元素的点击事件
          $("ul").on("click", "li", function(e) {
            console.log(e.target)
          });
    ```

# 4. 动画操作

- 初体验
  - animate
  
    - ```react
          $('ul').animate({
              height: 'toggle',
              opacity: 'toggle'
            }, 3000, "linear",function() {
              console.log('动画完成')
            })
      ```
  
    - 4个参数,只有第一个properties必须的,其他如3000之后的可以省去
  
  - animate properties参数支持类型
  
    - 数值:height:0
  
    - 关键字:show,hide和toggle
  
    - 相对值: +=,-=
  
      - ```react
            height: '+=100px',//+=100px表示相对于原来的高度增加100px
                  opacity: '-=0.5'//-=0.5表示相对于原来的透明度减少0.5
        ```
  
    - em,%(注意父元素要有高度,因为相对的父元素)
  
  - 手写一个隐藏显示和切换





- hide show toggle

  - ```react
    $('ul').click(function(){
    	$('.box').hide()或.show()或.toggle({
    		duration:3000,
    		complete: function() {
    			clg	
    		}
    	})
    })
    ```

  - 

- 动画队列
  - 调用animate delay这些函数会放入动画队列中执行

  - queue

    - ​    console.log($('ul').queue())

  - stop

    - ```react
          $('ul').animate({
              top:100
            },2000)
            $('ul').animate({
              left:100
            },2000)
            $('ul').animate({
              top:0
            },2000)
            $('ul').animate({
              left:0
            },2000)
      ```

    - ```react
      // $('ul').stop()//停止当期的动画队列,还会执行下一个动画队列
              //默认值$('ul').stop(false,false)
      
              // $('ul').stop(true)//停止当期的动画队列,不会执行下一个动画队列
              $('ul').stop(true,true)//立刻执行完当前动画队列,同时删除其他动画队列
      ```

# 5.遍历jQuery的对象

- jQuery对象中的each
- jQuery函数中的each

```react
  //1.使用jQuery对象的each方法遍历所有li元素
      $('ul li').each(function(index,Element){
        console.log(index,Element)
      })

      //2.使用jQuery函数的each方法遍历所有li元素
      $.each($('ul li'),function(index,Element){
        console.log(index,Element)
      })
```
